Build a modern AI-powered web application called "Script to Image Finder".

Objective:
The website should allow users to paste an entire script, automatically split it into individual lines, and generate highly relevant images for each line. Users should then be able to preview and download every image individually or download all images together.

Tech Stack:
- Frontend: React + TypeScript + Tailwind CSS
- Backend: Node.js + Express
- Image API: Google Custom Search API OR Bing Image Search API OR Unsplash API (Make the API provider easily configurable)
- Optional AI: OpenAI API for improving search queries before fetching images.

UI Requirements:
Create a clean, premium, responsive interface similar to ChatGPT or Notion.

Sections:
1. Header
   - Logo
   - Website Name
   - Dark/Light Mode Toggle

2. Script Input
   - Large textarea
   - Placeholder:
     "Paste your script here..."
   - Character Counter
   - Clear Button

3. Controls
   - Generate Images Button
   - Stop Button
   - Progress Indicator
   - Total Lines Counter

Workflow:
When the user clicks "Generate Images":

1. Read the entire script.
2. Split the script into separate lines.
3. Ignore empty lines.
4. Number every line.
5. For every single line:
   - Convert it into a natural image search query.
   - Remove punctuation.
   - Improve the search query using AI if enabled.
   - Search for relevant high-quality images.
   - Fetch at least 4 images per line.
6. Display the results one line at a time.
